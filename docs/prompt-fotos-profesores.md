# Prompt para Fotos de Profesores - Farray's Dance Center

## Guía de Edición con Nano Banana (Google Gemini)

### Objetivo

Crear fotos de equipo consistentes y profesionales nivel enterprise, donde todos los profesores se vean con el mismo encuadre, tamaño de cara y estilo visual.

---

## Prompt Principal (Español)

```
INSTRUCCIONES CRÍTICAS - ENCUADRE UNIFORME:

1. RECORTE Y COMPOSICIÓN:
   - Encuadre tipo "headshot corporativo": desde la parte superior
     de la cabeza hasta mitad del pecho aproximadamente
   - La cara debe ocupar el 40-50% del alto de la imagen
   - Centrar la cara horizontal y verticalmente
   - Si es necesario recortar brazos o cuerpo, hazlo - lo importante
     es que la cara tenga el mismo tamaño y posición que las demás
   - Aspecto cuadrado (1:1) o retrato (3:4)

2. NO MODIFICAR LA PERSONA:
   - Mantén exactamente la cara, expresión y rasgos sin cambios
   - No alterar tono de piel, facciones ni expresión

3. FONDO:
   - Degradado radial de negro puro (#000000) en los bordes
     hacia rojo burdeos muy oscuro (#800020) en el centro
   - El degradado debe ser muy sutil, casi imperceptible
   - Añadir un halo de luz difuso rojo carmesí (#B01E3C)
     detrás de la cabeza/hombros

4. ILUMINACIÓN:
   - Luz principal suave desde arriba-frontal (45°)
   - Rim light sutil con tono carmesí (#B01E3C) en el
     contorno de hombros y cabello
   - Sin sombras duras en la cara
   - Iluminación favorecedora y uniforme

ESTILO FINAL: Fotografía corporativa premium de academia de danza
de alto nivel. Aspecto cinematográfico, elegante y cohesivo.
```

---

## Prompt Corto (alternativa)

```
Recorta como headshot corporativo: cara centrada ocupando 40-50%
del alto, desde cabeza hasta mitad de pecho. Puedes cortar brazos/cuerpo.

Mantén la cara exactamente igual, solo cambia el fondo.

Fondo: degradado sutil de negro (#000000) a burdeos oscuro (#800020).
Halo de luz carmesí (#B01E3C) detrás de la persona.
Iluminación de estudio profesional con rim light carmesí sutil.

Resultado: headshot enterprise uniforme y premium.
```

---

## Prompt para Igualar con Foto de Referencia

Una vez tengas la primera foto perfecta, usa este prompt para las demás:

```
Usa la primera imagen como referencia exacta de estilo.

Para la segunda imagen:
1. Recorta para que la cara tenga EXACTAMENTE el mismo tamaño
   y posición que en la referencia
2. Aplica el mismo fondo, degradado e iluminación
3. Mantén la cara de la persona sin modificar

El resultado debe verse como parte del mismo set de fotos corporativas.
```

## Especificaciones Técnicas

### Colores de Marca

| Color          | Hex       | Uso                      |
| -------------- | --------- | ------------------------ |
| Carmesí (logo) | `#B01E3C` | Rim light, halo, acentos |
| Burdeos oscuro | `#800020` | Centro del degradado     |
| Negro puro     | `#000000` | Bordes del fondo         |

### Dimensiones Recomendadas

| Formato   | Tamaño     | Uso                  |
| --------- | ---------- | -------------------- |
| Cuadrado  | 800x800 px | Tarjetas, grid       |
| Retrato   | 600x800 px | Páginas de equipo    |
| Thumbnail | 400x400 px | Listados, navegación |

### Checklist de Consistencia

Antes de dar por finalizada cada foto, verificar:

- [ ] Cara ocupa ~40-50% del alto de la imagen
- [ ] Cara centrada horizontal y verticalmente
- [ ] Mismo degradado de fondo que las demás
- [ ] Rim light carmesí visible en hombros/cabello
- [ ] Iluminación facial similar a las otras fotos
- [ ] Sin distorsión de rasgos faciales

---

## Flujo de Trabajo Recomendado

1. **Editar la primera foto** hasta que quede perfecta
2. **Guardarla como referencia** visual
3. **Editar las demás** usando el "prompt para igualar"
4. **Comparar lado a lado** antes de exportar
5. **Exportar todas** al mismo tamaño final

---

## Notas

- Las imágenes generadas con Nano Banana incluyen marca de agua SynthID (invisible)
- Google no penaliza en SEO por usar imágenes editadas con IA para fondos
- Mantener las caras originales intactas es clave para autenticidad
