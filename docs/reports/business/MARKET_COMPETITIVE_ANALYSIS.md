### Análisis de Pricing Strategy

| Aspecto                  | Farray's         | Dance Emotion      | Así Se Baila     |
| ------------------------ | ---------------- | ------------------ | ---------------- |
| **Posicionamiento**      | Premium          | Accesible          | Medio            |
| **Modelo**               | Cuotas mensuales | Créditos flexibles | Tarifas variadas |
| **Flexibilidad**         | Media            | Alta (rollover)    | Alta             |

### Hallazgos Clave:

1. **Farray's es 15-20% más caro** en planes regulares
2. **Plan ilimitado de Farray's es 2x** el de Dance Emotion (300€ vs 150€)
3. **Sistema de créditos** de Dance Emotion es más flexible (rollover)
4. **Farray's justifica precio** con clases "Premium" con Yunaisy

**Riesgo:** Precio alto puede ser barrera para nuevos alumnos sensibles al precio

✅ FORTALEZAS:

❌ DEBILIDADES:
- Podría sonar abstracto para algunos
- Menos enfocado en resultados tangibles
```

#### Dance Emotion

```
✅ FORTALEZAS:
- "Empoderamiento" es palabra potente

- Mensaje inclusivo ("Los que lo hicieron no se arrepienten")

❌ DEBILIDADES:
- Menos diferenciado (muchas escuelas usan "empoderamiento")
- Mensaje genérico de danza
```

#### Así Se Baila

```
✅ FORTALEZAS:
- "Donde los sueños se bailan" es poético
- Reconocimiento oficial destacado
- Tono aspiracional

❌ DEBILIDADES:
- Mensaje más tradicional
- Menos gancho emocional directo
